# Guia de estudio - Capitulo 6

Guia de estudio y preparacion para la evaluacion del Capitulo 6 de la tesis de maestria: **Extension no lineal mediante computacion de reservorio estructurado**.

## Contenido

- Conceptos clave con analogias y diagramas
- Arquitectura del modelo TCROC-SSRC
- Resultados experimentales con 14 figuras
- Interpretacion detallada de figuras clave
- 18 preguntas de evaluacion con respuestas
- Puntos criticos y frases de respuesta
